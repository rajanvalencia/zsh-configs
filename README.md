# zsh-configs

My terminal configurations for MacOS

## Installation

1. Install [Homebrew](https://brew.sh/)

2. Install

```bash
make install-all
```

3. Change terminal font

## Help

```bash
make help
```

## Remove

```bash
make remove-all
```
